# cli-ls

## Installation

```
git clone https://github.com/aziascreations/cli-ls.git
cd cli-ls
npm i
node ./
```
I haven't figured out how to install it globally.

## Usage

See "man ls"

## Requirements

#### Core modules
* fs
* path

#### External Modules
* command-line-args
* command-line-usage
